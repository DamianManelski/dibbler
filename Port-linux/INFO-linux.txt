
   Dibbler - a portable DHCPv6
  -----------------------------
  authors: Tomasz Mrugalski <thomson@klub.com.pl>
           Marek Senderski  <msend@o2.pl>
           
  Compilation info
 -----------------------
  Edit Makefile.inc to suit your needs.
  You need flex and bison++ to generate parser files. 
  bison++ will be included in next release.

  Then just type "make" and have your fingers crossed.
 
  Users guide
 ------------------------
  Make sure there is a directory /var/lib/dibbler.
  Extract this archive to /var/lib/dibbler
  
  Server start: ./dibbler-server-linux
  Server start in debug mode: ./dibbler-server-linux -d
  Server stop: kill `cat server.pid`

  Client start: ./dibbler-client-linux
  Client start in debug mode: ./dibbler-client-linux -d
  Client stop: kill `cat server.pid`
